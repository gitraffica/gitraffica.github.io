Attribute Forge

Attribute Forge is a lightweight yet powerful system that allows developers to create flexible, conditional buffs and inject dynamic behaviors seamlessly into their Unity projects.

  • Zero-coupling Buff Implementation
  • Timed Modifier System
  • Conditional Buff Support
  • Behavior Injection and MonoBehaviour Integration
  • Performance Optimized and Priority System

Whether you're developing an RPG, skill-based system, or managing complex gameplay mechanics, Attribute Forge provides an elegant and powerful solution.
